# 1º Trabalho Computacional - Sinais e Sistemas (PEN2244)

Este repositório contém a resolução do primeiro trabalho computacional da disciplina **PEN2244 - Sinais e Sistemas**, ministrada pelo **Prof. Pedro Thiago Valério de Souza** no Centro Multidisciplinar de Pau dos Ferros da **Universidade Federal Rural do Semi-Árido (UFERSA)**.

A resolução aborda de forma analítica e numérica problemas fundamentais de sinais de tempo contínuo, modulação em amplitude, sistemas lineares e invariantes no tempo (LTI) e modelagem de circuitos elétricos via equações diferenciais ordinárias.

## Metodologia e Ferramentas

O trabalho foi desenvolvido utilizando a linguagem de programação **Python** em ambiente **Jupyter Notebook**. A implementação prioriza a comparação entre métodos analíticos e simulações numéricas, fazendo uso das seguintes bibliotecas:

* **NumPy**: Processamento de arrays multidimensionais e funções matemáticas.
* **Matplotlib**: Geração de representações gráficas de sinais e respostas de sistemas.
* **SymPy**: Computação simbólica para dedução de expressões matemáticas e resolução de EDOs.
* **SciPy**: Ferramentas para integração numérica e convolução de sinais.
